Sum of Palindromes

An implementation of a research paper by Javier Cilleruelo, Florian Luca and Lewis Baxter proves that any positive integer can be expressed as a sum of three palindrome numbers. The proof is in a form of 42 page algorithm which is implemented in this program.

The program will ask you to enter a number and display the three palindromes. A flask based web version is also created and the steps to execute are mentioned in the github itself. There is also a test which will compare this algorithm with brute force approach and verifies the correctness.
